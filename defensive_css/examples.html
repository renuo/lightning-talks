Flexbox Wrapping

When there is not enough space, child items won't wrap into a new line by default.


  .options-list {
    display: flex;
  }



.options-list {
  display: flex;
  flex-wrap: wrap;
}

Image distorsion

When we don't have control over an image's aspect ratio on a web page, it's better to think ahead and provide a solution when a user uploads an image that isn't aligned with the aspect ratio.


  img {
    object-fit: cover;
  }
